What git command creates a branch?

What is the difference between a fast-forward and recursive merge?
answer: fast forwards can only happen if there have not been commits on the original branch while the new branch is being worked on

What git command changes to another branch?
answer: git checkout Branch_Name

What git command deletes a branch?
answer:git branch -d Branch_Name or git branch -D Branch_Name

answer: When there have been different commit changes to the same file, Git gets confused and doesn't know which to merge
